# Fake News Detection using Machine Learning

This project detects fake news statements using NLP and machine learning.

## Technologies Used
- Python
- scikit-learn
- TF-IDF
- Streamlit

## Model
Logistic Regression classifier trained on labeled dataset.

## Run the Application

Install dependencies:

pip install streamlit scikit-learn pandas joblib

Run the app:

streamlit run app.py

## Features
- Fake news classification
- Streamlit web interface
- Model evaluation metrics
